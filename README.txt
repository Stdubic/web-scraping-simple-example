simple web scrapping app
========================

Basic code for scraping latest comments from "http://www.dailymail.co.uk/sciencetech/article-2986641/Now-really-time-Apple-finally-unveils-Watch-comes-three-models-boasts-18-HOURS-battery-life.html"

#Installation

- install node.js

- npm install

#How to run
- node node.js
